## 🚀 About The Project

An intelligent AI-powered chatbot that helps users discover products, compare options, and get personalized shopping recommendations in real-time using Natural Language Processing.

## 🧠 Features

- 🤖 AI-powered product recommendations  
- 💬 Conversational chatbot interface  
- 🔍 Smart search using NLP  
- 📊 Personalized suggestions  
- ⚡ Fast & interactive UI  
- 🧠 Intent-based query understanding

## 🎯 Overview

This project is a **Machine Learning-based Spam Detection System** that classifies messages into:

* 🚨 Spam
* ✅ Ham (Not Spam)

Built using:

* 🧠 TF-IDF (feature extraction)
* 🤖 Naive Bayes (model)
* 🌐 Streamlit (web interface)
